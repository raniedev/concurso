# Histórico e Evolução da Web

A World Wide Web é um sistema informações interligados por hipertextos (texto, imagens, vídeos, áudios e vários outros tipos de arquivos), cujo o acesso se dá por meio de software denominados navegadores (browsers). A Web é considerada a teia mundial de informações.
## Web 1.0 (Read Only, Static)

A Web é estática e apenas possui leitura de textos, sem interação com o o usuário que apenas consome o conteúdo.
**Tecnologias Fundamentais:** HTML Puro
## Web 2.0 (Read, Write)

A Web se torna colaborativa, com pessoas criando conteúdos (Search, Tag Collaborate).
**Algumas Tecnologias Fundamentais:** XML, PHP, AJAX, RSS, ASP, JSON, REST, SOAP, Java EE, MySQL
## Web 3.0 (Read, Write, Understand)

A Web passa a ter significado através da sua semântica, e começa a ser compreendida por algoritmos e robôs.
**Algumas Tecnologias Fundamentais:** RDF, XHTML, RDFS, OWL, SPARQL, JSON-LD, RESTful API, GraphQL, Big Data (Hadoop, Spark), IPFS
## Web 4.0 (Read, Write, Execute)

É a próxima geração da internet, também conhecida como **Web Inteligente ou Simbiótica (Symbiotic Web)**, caracterizada pela integração profunda da inteligência artificial, dispositivos conectados e experiências imersivas. 
**Tecnologias Fundamentais:** Artificial Intelligence (AI), Machine Learning (ML), Augmented Reality (AR), Virtual Reality (VR), Internet of the Things (IoT). 
## Web 5.0 (Read, Write, Feel)

Em desenvolvimento está a Web capaz de reconhecer e responder a emoções humanas, através de reconhecimento sensorial, também chamada de **Web Sensorial (Sensory Web)**.
