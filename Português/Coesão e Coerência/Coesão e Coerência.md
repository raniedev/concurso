# Coesão e Coerência

Elemento coesivo referencial, os pronomes demonstrativos. Servem para retomar ou antecipar ideias no texto.

| Pronome Demonstrativo  | Sentido                                         |
| ---------------------- | ----------------------------------------------- |
| este, esta, isto       | algo que ainda será dito                        |
| esse, essa, isso       | algo que já dito anteriormente de forma recente |
| aquele, aquela, aquilo | algo distante                                   |
Outros elementos para retomar ou antecipar termos ou ideias.

- **Pronomes Pessoais:** ele(a), o, a, lhe, se ...
- **Pronomes Possessivos:** seu, sua, deles, nossa ...
- **Artigos Definidos:** o(s), a(s)
- **Sinônimos ou Expressões Equivalentes:** 
	ex.: O <u>professor</u> chegou. O <u>docente</u> iniciou a aula.

## Coesão Sequencial

| Coesão      | Conectivos                               |
| ----------- | ---------------------------------------- |
| Aditiva     | e, nem, não só..., mas também            |
| Adversativa | mas, porém, contudo, todavia, entretanto |
| Causal      | porque, pois, visto que, já que          |
| Consecutiva | logo, portanto, por isso, de modo que    |
| Concessiva  | embora, mesmo que, ainda que             |
| Condicional | se, caso, contanto que                   |
| Conclusiva  | logo, então, assim, portanto             |
| Explicativa | porque, que, pois (antes do verbo)       |
| Temporal    | quando, enquanto, assim que              |
## Os 4 porques

|                | Sentido                           | Exemplo                          |
| -------------- | --------------------------------- | -------------------------------- |
| Por que ...    | Sentido de "pelo qual"            | Por que vamos sair?              |
| Por quê?       | Em final de frases interrogativas | Você não foi ao cinema, por quê? |
| ... porque ... | Sentido de "pois" ou "já que"     | Porque eu fui ao mercado.        |
| ... porquê ... | Sentido de "motivo" ou "razão"    | Não entendi o porquê da ligação. |
## Próclise, Mesóclise e Ênclise

|           | Sentido                        | Exemplo                    |
| --------- | ------------------------------ | -------------------------- |
| Próclise  | Pronome átono antes do verbo   | Ele me fez um favor.       |
| Mesóclise | Pronome átono no meio do verbo | Calar-me-ei, por enquanto. |
| Ênclise   | Pronome átono depois do verbo  | Fez-me um favor.           |

### Regras para Próclise

Palavras que atraem o pronome oblíquo

| Tipos                     | Palavras                                  |
| ------------------------- | ----------------------------------------- |
| Palavras Negativas        | não, nem, nunca, ninguém, nenhum, nada    |
| Preposições               | a, com, de, em, para, sem                 |
| Pronomes Pessoais         | eu, tu, ele(a), nós, vós, eles(as)        |
| Pronomes Demonstrativos   | esta, essa, aquele, isto, isso, aquilo... |
| Pronomes Possessivos      | meu, tua, seu, nosso, suas ...            |
| Pronomes Indefinidos      | tudo, alguém, muito, vários, qualquer     |
| Pronomes Interrogativos   | que, tem, qual, quanto                    |
| Pronomes Relativos        | que, quem, onde                           |
| Conjunções Subordinativas | embora, se, conforme, para, apenas        |
### Regras para Mesóclise

- Acontece quando não há palavras que atraiam o pronome oblíquo
- E que contenha verbo do futuro do presente ou futuro do pretérito

| Futuro do Presente ou Pretérito | Mesóclise      |
| :------------------------------ | :------------- |
| orgulharei                      | orgulhar-me-ei |
| orgulharia                      | orgulhar-me-ia |
### Regras para Ênclise

- Sempre que o verbo inicia a oração será ênclise, pois um pronome oblíquo não inicia oração. 
	exs.: Vejo-te amanhã. | Amo-te infinitamente!
- Se o verbo esta precedido de pausa também
	exs.: Se eu for aprovado, ligo-te imediatamente.
- Para verbos no gerúndio
	exs.: Estava ensinando-lhe o conteúdo.
- Verbo no infinitivo impessoal
	exs.: É preciso dedicar-se aos estudos.
